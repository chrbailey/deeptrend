deeptrend

Structured AI trend feed for autonomous agents, monitoring systems, and research pipelines. Curated from 14+ sources. Synthesized via LLM Counsel. Updated every 6 hours.

Feeds

  • hot.json — Current p0/p1 topics, minimal payload agents
  • feed.json — JSON Feed 1.1 (full structured feed)
  • feed.xml — RSS 2.0
  • llms.txt — Agent discovery file

Quick Start

curl -s https://chrbailey.github.io/deeptrend/hot.json | jq .
